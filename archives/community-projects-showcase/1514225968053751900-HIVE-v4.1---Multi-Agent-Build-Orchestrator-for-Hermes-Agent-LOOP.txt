    One sentence → automated build → iterable artifact.
    
    How it works:
                         ┌──────────────────┐
                         │  "make a calc"    │
                         └────────┬─────────┘
                                  ↓
                        ┌──────────────────────┐
                        │     AGENT LOOP       │
                        │                      │
                        │  ARCHITECT → brief   │
                        │  PLANNER   → DAG     │
                        │  CODER × 3 → code    │
                        │  TESTER    → verify  │
                        │  REVIEWER  → 4-layer │
                        │  TOOLMAN   → ship    │
                        │        ↓             │
                        │  ┌──────────────┐    │
                        │  │ issues found │────│──→ back to CODER
                        │  └──────────────┘    │
                        │  ┌──────────────┐    │
                        │  │ tests fail   │────│──→ back to CODER
                        │  └──────────────┘    │
                        │  ┌──────────────┐    │
                        │  │ user not     │────│──→ iterate again
                        │  │ satisfied   │    │
                        │  └──────────────┘    │
                        └──────────────────────┘
                                  ↓
                          shippable artifact
    
    6 specialized agents in a self-correcting loop — review failures and test failures loop back to the coder pool automatically. User can also trigger iteration on the final result.
    
    15 MCP tools, 152 tests, Dashboard, rollback, cross-project memory.
    
    Built on Hermes Agent MCP — every agent call is a JSON-RPC exchange. Fully MIT open source.
    
    🔗 https://github.com/Ebonyhtx/Hermes-Hive
