    ### Streaming
    ```python
    stream = client.chat.completions.create(
        model="vibe",
        messages=[{"role": "user", "content": "Explain quantum computing"}],
        max_tokens=2000,
        stream=True
    )
    for chunk in stream:
        if chunk.choices[0].delta.content:
            print(chunk.choices[0].delta.content, end="")
    ```
    
    ### Key Parameters
    | Parameter | Value | Notes |
    |-----------|-------|-------|
    | `model` | `"vibe"` | Any string works, it ignores it |
    | `max_tokens` | 2000+ | Model thinks first, needs room |
    | `temperature` | 1.0 | Recommended for reasoning |
    | `top_p` | 0.95 | Recommended |
    | `stream` | true/false | SSE streaming supported |
