    An Open-Source Framework for Local Static Analysis & Semantic AI Code Auditing
    
    What is Hermes AI & How Does It Work?
    Hermes AI is an open-source Developer Tool (DevTool) engineered as a dual-engine autonomous cybersecurity agent. Built strictly for software architecture verification and vulnerability research, it allows developers and systems engineers to audit source code structures in real-time through a secure interface:
    
    Layer 1 (Local Static Analysis Engine): A zero-cost, highly optimized local filtering layer that instantly scans code modules to flag notorious structural flaws, improper authorization boundaries, and control-loss vectors.
    
    Layer 2 (AI Auditor Agent): A deep semantic auditing layer powered by google/gemini-2.5-flash via OpenRouter. It comprehensively evaluates abstract business logic, access control hierarchies, and functional reentrancy risks.
    
    Smart Fix (Autonomous Patching): Moving beyond traditional static analysis tools that only identify errors, Hermes AI's auto-patching engine autonomously generates and embeds optimized, secure software code patches directly beneath each detected vulnerability.
    
    Technical Deliverables & Downstream Artifacts
    
    - Professional PDF Security Report: The engine compiles a comprehensive, pixel-perfect PDF report displaying execution timestamps, code module identifiers, and an official Security Score (0 to 100) computed by the semantic agent.
    
     Severity Categorization: All structural anomalies within the document are labeled following strict industry software security protocols (Critical, High, Medium, Low).
    
     Implementable Patches: Complete, fully refactored code snippets are safely embedded into the document for immediate optimization.
