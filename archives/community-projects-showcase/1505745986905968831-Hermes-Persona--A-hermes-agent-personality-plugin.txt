    I built a super light weight Hermes Agent plugin that gives your agent a persistent character identity. It survives sessions, restarts, and context resets. It's a five-layer personality stack:
    
    1. **SOUL.md** — core identity (who your character is)
    2. **/personality** — how they think, feel, and speak
    3. **Prefill** — few-shot behavioral anchor (survives context drift)
    4. **persona-config.json** — dynamic rules, time-slots, relationship dimensions <- my plugin works on this layer
    5. **Memory** — what they learn across sessions (MCP memory providers)
    
    Changes hot-reload on save. No restart needed. 🔁
    
    The repo ships with example configs so you can see how it works immediately — including a black cat named Luna who ends every sentence with "nya~" 🐱
    
    **Running in 30 seconds from install:**
    
    ```
    hermes plugins install kenyonxu/hermes-persona --enable
    ```
    
    Repo: https://github.com/kenyonxu/hermes-persona
    
    Docs in CN + EN. Pick a character or build your own — the five layers fit anything.
