    hey everyone, wanted to share a project i’ve been building lately:
    
    cammander
    https://github.com/GuideboardLabs/cammander
    
    it’s a portable browser based AI IDE designed around terminal first workflows
    
    features currently include:
    
    live terminal
    workspace aware AI chat
    file explorer
    code editor with syntax highlighting
    image viewer
    mobile friendly UI
    multi provider model support
    
    the whole idea was to recreate the feeling of Claude Code / Codex style workflows without being locked into a heavyweight desktop app or single provider ecosystem
    
    it runs in the browser, works across devices, and is designed to stay fast + hackable
    
    i’m actually using it daily now instead of my old VS Code + terminal + AI stack, and i’m actively using it to build Oathweaver after having to rebuild parts of that backend architecture from scratch
    
    roadmap before RC includes:
    
    llama.cpp
    vLLM
    LM Studio
    Codex integration
    graph + chart generation from codebases
    Obsidian style memory vault for long term project context
    
    still early but usable today and moving quickly
    
    would love feedback from anyone into AI tooling, terminal workflows, local models, or FOSS dev environments
