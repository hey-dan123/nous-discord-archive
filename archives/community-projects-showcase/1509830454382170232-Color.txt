    Built Colors — an open-source AI agent with a structural fix for indirect prompt injection.
    The core idea: tag every instruction with it source (user / agent / external_content) and block external content from triggering ool calls at the execution gate. Normal agents can't distinguish between a user instruction and a malicious instruction embedded in a webpage they're reading. Colors blocks that architecturally, not through prompt filtering.
    Live demo: huggingface.co/colour-foundation-ai/Colors-security-gate
    Research + code: github.com/thecolourfoundation/Color
    Also opened a proposal to bring this upstream to Hermes: https://github.com/NousResearch/hermes-agent/issues/33997
    Would love feedback from anyone who's thought about the injection attack surface."
