    **AgentBuddy** is more than just a desktop UI for Hermes.  
    It’s a local-first, end-to-end AI automation platform / sandbox  
    
    ⬇️ Download https://github.com/spankyed/AgentBuddy-releases/releases
    
    ## ⚡ Getting Started
    1. Download AgentBuddy
    2. Go through onboarding
    3. Select “Yes, setup Hermes” during onboarding
    
    ## ✨ Features
    ### AI & Automation 🤖
    - Hermes Plugin & integration
    - Claude Code integration
    - Create custom workflows + actions
    - Build your own `/commands` for automations & tooling
    ### Virtual Library System 📚
    - Virtual documents & folders
    - Link real filesystem folders / files directly into the app
    ### Full IDE Experience 🛠️
    - Integrated code editor
    - Built-in terminal
    - Git integration
    - View/create/update PRs from inside the app
    - Manage commits, stashes & switch worktrees
    ### Productivity 📝
    - Notes & Todo lists
    - More tools and features with new ones constantly being added
    
    ## 🧪 Currently In Development  
    - Codex support
    - Linux support
    - LSP support in the IDE
    - Hermes Kanban integration
    - System for creating and finding plugins
      - build your own custom UIs!
