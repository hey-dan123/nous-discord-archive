    I was tired with chrome and firefox's default pdf viewer, i built this with hermes with deepseek v4 flash.
    
    the app have some cool features like
    
    1. Multiple reading layouts
    2. multiple reading themes
    3. importing papers directly from url.
    4. progress tracking, tags, folder, annotation
    5. supports local models with ollama cloud
    6. artifacts which shows all kinds of links from pdf into a good place to skim through/
    
    and it's available for windows and linux system to download here
