    Hi all sharing what I shipped tonight inside Hermes Workspace: a real multiplayer 3D RPG called HermesWorld (+ Multiplayer )
    The pitch: docs are boring, agents are abstract. So instead of writing another tutorial, I turned Hermes Agent onboarding into a game world you walk around in. 
    
    Talk to NPCs, complete quests, equip knight gear, meet other live builders standing in the same world as you.
    
    How it was built:
    
    Started as a 2D top-down sketch generated by Hermes itself. Then a Kimi-driven swarm of agents took it from 2D → 3D in stages, each agent owning a system:
    ⌘ 2D scaffold + display-name flow + avatar customizer
    ⌘ Full 3D world (R3F + Three.js) — walking player, NPCs, click-to-walk
    ⌘ Combat (Strike / Dash / Bolt / Summon), action bar, HUD, minimap
    ⌘ 5-step Hermes tutorial chain → Forge Gate unlock
    ⌘ Web Audio synth ambient + per-world voice narration (no API key needed)
    ⌘ Multiplayer over Cloudflare Durable Objects with HTTP polling fallback
    ⌘ Knight armor + Hermes statues + braziers + stone-tile plazas
    ⌘ Diplomacy + Summoning quests so all 6 Hermes skills are playable
    
    The 6 Hermes skills as game mechanics:
    ⚔ Promptcraft — Forge build dialogs
    ⚔ Worldsmithing — Forge entry + Forge Shard quest
    ⚔ Engineering — Trainer's Ring + Glitch Wisp combat
    ⚔ Oracle — Oracle Temple riddle
    ⚔ Diplomacy — Agora quest: meet a real live builder + chat with them
    ⚔ Summoning — Action bar key 4: spawn a glowing Hermes familiar that orbits you for 60s (sub-agent orchestration)
    What's in it:
    ✦ 6 worlds: Training Grounds → Agora Commons → Forge → Grove → Oracle Temple → Benchmark Arena
    ✦ 6 enterable buildings in Agora (Tavern, Bank, Smithy, Inn, Apothecary, Guild Hall)
    ✦ 9-preset Sims-lite avatar customizer with skin/hair/outfit/cape/helmet/weapon
    ✦ Full knight armor: cuirass, winged Hermes sigil, tasset, gauntlets, greaves
    ✦ Live multiplayer via global Cloudflare Worker hub
    ✦ Per-world voice narration that auto-plays on entry
    ✦ ASCII NPC portraits in dialog cards (pyfiglet via the AgentSkill)
    ✦ Cinematic camera mode (Tab cycles 6 preset angles for filming)
    ✦ Premium HermesWorld branding — gold serif title, animated agent network, lore-quote loading screens
    
    Built solo over a sprint with Hermes Workspace running multiple agent personas in parallel. The whole thing lives at /playground inside Workspace — no installs, no accounts. Just a display name and you're in.
