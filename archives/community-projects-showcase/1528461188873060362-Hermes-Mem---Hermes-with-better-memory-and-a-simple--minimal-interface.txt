    Hermes Mem is an independent community fork of Hermes Agent with improved memory and a simpler, more minimal desktop interface.
    
    It installs separately from your existing Hermes installation, uses its own folder, and does not modify your regular Hermes setup.
    Hermes Mem uses three types of memory:
    10-day memory — detailed recent context is kept for 10 days and then gradually forgotten.
    Permanent memory — only important information that should always be remembered is stored here.
    Events — temporary situations with their own time period, so the AI can understand what is happening now and avoid mixing up past and current information.
    
    You can explore the source code, read more about the project, or ask questions and report bugs through GitHub Issues:
    **https://github.com/FPP-Labs/hermes_mem**
    Install on Linux or macOS:
    **git clone https://github.com/FodorProPro/hermes_fpp.git && cd hermes_fpp && ./install.sh**
    When the installer menu appears, choose 1.
