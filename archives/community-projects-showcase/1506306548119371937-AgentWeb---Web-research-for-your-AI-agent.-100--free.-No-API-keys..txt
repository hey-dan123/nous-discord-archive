    I was tired of paying Browserbase/ Firecrawl, or juggling 4 different free API keys. For a basic thing — search the web, get results, move on.
    
    So I built AgentWeb. It calls 12+ free sources (DDG, Jina, GitHub, HN, Reddit, StackExchange, Wikipedia, arXiv, YouTube, Twitter/X, Nominatim...) in parallel, ranks results with BM25 + a tiny ONNX model on your CPU (FlashRank), and outputs clean JSON. Research modes decompose your query, extracts evidence, detects contradictions, writes a report. All local. All free.
    
    3 modes
    
    **search**: Fires all providers, short-circuits when it has enough, ranks semantically. Takes ~2-8s. `--prefer github / --exclude youtube / --context "whatever"` for your Hermes to steer sources based on job requirements.
    
    **research**: Search + fetch + structured evidence pack with coverage scores. Only `--context` for steering (no prefer/exclude here). ~30-60s.
    
    **deep-research**: Multi-branch decomposition, per-branch source hints, contradiction detection, full structured report. Also `--context` only. ~60-120s.
    
    All three output JSON (or md) with quality scores and provenance. Built to be controlled by your Hermes Agent, not a human clicking buttons.
    
    > `agentweb search "MCP server implementation" --prefer github --exclude youtube,arxiv
    > agentweb deep-research "Claude Code vs Cursor pricing" --context "dev tooling budget" -o report.md
    > `
    
    One command install:
    
    `curl -fsSL https://raw.githubusercontent.com/thanhan-a17/agentweb/main/install.sh | bash`
    The Hermes skill loads automatically after install, ready to use.
