    🚀 **Cronalytics v1.0 is live**
    
    A dashboard plugin for Hermes Agent that turns your cron job cost from a mystery into a number you can act on.
    
    ---
    
    **What it does**
    
    Cronalytics hooks into `on_session_end`, captures every cron-originated run, and writes cost, tokens, model, and duration into a local fact database. A reconciliation scanner backfills historical data so nothing is lost.
    
    All of it surfaces in a dedicated `/cronalytics` tab inside the Hermes dashboard.
    
    ---
    
    **Dashboard surface**
    
    • **Summary Board** — total runs, cost, tokens, and pace multiplier with vs-prior-period deltas
    • **Leader Board** — spotlight cards for top runs, top cost, top tokens, and highest pace (your early-warning signal)
    • **Per-Model Breakdown** — proportional cost bars so you can spot model bloat at a glance
    • **Jobs Breakdown** — 8-column sortable table with expandable detail rows showing token splits, schedule metadata, success/failure breakdown, and a "See Runs" modal
    • **Outcome toggle** — filter All / Success / Failure with conditional card colors
    • **Mode toggle** — filter All / Agent / No agent (script jobs are finally visible)
    • **Sync Now** — manual backfill with completion toast
