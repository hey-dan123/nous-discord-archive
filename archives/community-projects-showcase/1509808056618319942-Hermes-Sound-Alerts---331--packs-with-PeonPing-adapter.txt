    I built a Hermes Agent plugin for PeonPing: Sound Alerts for Hermes CLI.
        
    PeonPing is a local sound/voice-pack system, built for other harnesses. I discovered it recently, immediately wanted it inside Hermes, and since Hermes support didn’t exist yet, I made an adapter + standard Hermes plugin.
        
    What it does:
    - plays PeonPing sound/voice-pack events for Hermes lifecycle hooks
    - supports session start/end, turn completion, approval prompts, selected tool progress, terminal failures, and subagent completion
    - adds a Sound Alerts dashboard tab for browsing/installing/switching/previewing packs
    - fails open if peon isn’t installed, so Hermes won’t break
        
    If you don’t have PeonPing installed yet, the README includes install instructions for macOS and Linux/WSL.
        
    I kept the dashboard aligned with Hermes’ existing design and tuned it from there. I personally built this because I wanted custom voice packs for my own Hermes setup, and it’s been surprisingly fun giving the CLI a more present “voice” while working.
        
    Repo:
          https://github.com/brucestarlove/hermes-peonping
        
    Would love install reports, issues, PRs, and suggestions — especially from anyone trying it on devices I haven’t tested yet.
