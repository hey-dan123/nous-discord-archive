    Summary
    A native localization ecosystem for hermes-agent that automatically generates styled Turkish daily briefing cards. It pulls real-time market data, local crypto indices, and major news, delivering them directly via Telegram.
    
    Project Details & Tech Stack
    Data Core: Integrates BIST100 indices and local crypto token pricing in TRY using the CoinGecko API.
    
    News Aggregator: Dynamically filters daily briefs from major networks like Hürriyet, Bloomberg HT, and NTV.
    
    Dynamic Rendering: Python backend scripts compute the data models and leverage Pillow to render clean, embedded UI cards on the fly.
    
    Automation: Fully integrated via automated telegram cron schedules for morning deliveries.
    
    Structure: Built completely using standard .SKILL.md specifications to ensure stable prompt execution flows.
    
    Global Localization Docs
    Alongside the skill pack, I authored a comprehensive Localization Tutorial (#1423) to provide a blueprint for international developers looking to adapt and onboard localized skill packs or regional APIs into the ecosystem.

    Recent Stability Updates
    To ensure the automated delivery and local developer experience run seamlessly, I've recently pushed a few core fixes:
    
    Gateway & Telegram Patch (#25198): Fixed a critical bug where the agent permanently froze progress updates after hitting Telegram's flood control limits (RetryAfter). The logic has been successfully merged into main.
    
    OS Compatibility: Shipped macOS path fallbacks earlier, and currently have two active PRs open to unblock Windows environments—forcing UTF-8 subprocess outputs (#25200) and pinning Python versions in uv sync chains (#25555).
