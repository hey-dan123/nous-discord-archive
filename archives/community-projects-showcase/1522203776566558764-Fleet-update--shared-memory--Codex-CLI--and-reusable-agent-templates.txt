    Fleet is a local-first web console for creating, managing, coordinating, backing up, templating, and sharing memory across Dockerized Hermes agents on one or more trusted machines.
    
    Recent work adds three big pieces:
    
    Fleet shared memory: agents can keep their own private memory while also linking into a fleet-wide shared Mnemosyne store. Linked agents get dedicated fleet_shared_memory_* tools so they know when to use shared memory instead of local memory.
    
    Codex CLI capability: new Docker agents can be created with Codex CLI installed and authenticated using the same Fleet-managed OpenAI Codex device login.
    
    Template library: any local Docker Hermes agent can be captured as a reusable, secret-free template and redeployed multiple times to local or remote Fleet nodes. Templates strip secrets, Codex auth, bearer tokens, generated folders, and source-node shared-memory config, then rehydrate auth/capabilities from the target node.
    
    In short: Fleet can now create, coordinate, remember, clone, template, and redeploy agent setups much more safely and repeatably.
    
    Its available open source here: https://github.com/matt454/agent-fleet-console
