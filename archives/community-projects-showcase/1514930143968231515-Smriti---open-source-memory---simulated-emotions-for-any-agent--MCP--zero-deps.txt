    Hey everyone — just open-sourced Smriti, a portable memory + affect layer that works with any MCP-capable agent (built with Hermes in mind).
    
    It does the Hermes-style self-improvement loop (events → reflection → reusable skill docs) but framework-agnostic, plus temporal facts that actually supersede ("moved to Tokyo" beats "lives in London"), and a simulated emotion system where frustration escalates strategy (retry → switch → ask for help → cooldown) — never aggression. Values veto emotions, always.
    
    Benchmarked in CI: same 5 ops tasks, day-1 trial-and-error vs day-30 with memory → 5/5 skill recall, 75% fewer steps.
    
    Pure Python + SQLite, zero deps, MIT, 32 tests, 20 MCP tools.
