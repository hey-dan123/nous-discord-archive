    Built with React + TypeScript + Tailwind + Zustand. Connects to a local Ollama instance, streams responses, persists conversations. No backend.
        
        Brutalist design throughout — black background, monospace, 2px borders, zero radius, bracket-style buttons.
        
        Features:
        - Streaming responses with stop/retry
        - Conversation history with Zustand persist
        - Model selection from installed Ollama models
        - Connection status monitoring
        - Keyboard shortcuts
        
        Stack: React 18, TypeScript, Vite, Tailwind CSS, Zustand.
        
        Code: github.com/brutal-build/ollamachat
        
        Would appreciate any feedback.
