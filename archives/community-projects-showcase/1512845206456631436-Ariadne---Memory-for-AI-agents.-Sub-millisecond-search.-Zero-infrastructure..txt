    **Description:**
    I built a memory layer for AI agents that runs entirely locally with no daemon, no cloud, and no API keys. 311 tests, 0.03ms vector search via FAISS.
    
    The problem: every agent framework either uses ChromaDB (slow, no dedup), Mem0 (requires PostgreSQL + Qdrant), or rolls their own fragile SQLite tables. None of them handle the full lifecycle — storing, deduplicating, retrieving, evolving, and forgetting memories.
    
    What Ariadne does:
    - 0.03ms vector search (80x faster than ChromaDB)
    - Hybrid retrieval (BM25 + vector, fused with RRF)
    - Auto-deduplication via MinHash LSH (catches paraphrases)
    - Knowledge graph with BFS traversal
    - Memory lifecycle (Ebbinghaus forgetting curve)
    - Zero-config embeddings (auto-downloads ONNX model, works offline)
    - Agent-first — uses your agent's own LLM, no separate API key needed
    - 26 tool definitions for OpenAI function calling
    - Conversation summarization, temporal facts, entity resolution
    
    ```python
    from arriadne import AriadneMemory
    
    mem = AriadneMemory("memory.db")  # auto-detects embeddings
    mem.remember("VPS has 4 cores, 8GB RAM", importance=0.8)
    results = mem.recall("server specs", k=5)
    ```
