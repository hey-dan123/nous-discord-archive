    Hello all! 
    
    I already posted about this in chat but I figured that I'll also make a thread here. 
    
    In short, what I have for you today is a Vencord plugin that improves the markdown rendering experience inside of discord. Hermes (and any agent really) has a habit of returning markdown tables and lists in its replies. Discord will render basic markdown but has no support for tables, lists and other common MD features.
    
    My plugin patches Discord's markdown parser to also support these missing features. It leverages Vencord, a popular discord client mod, to do so. I attached some images of before and after applying this plugin.
    
    https://github.com/pnivek/BetterMarkdown
    
    I hope you all find it as useful as I have 🙂
