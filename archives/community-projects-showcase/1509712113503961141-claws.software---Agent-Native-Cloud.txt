    Built the cloud layer that AI agents can actually use on their own.
    
    claws.software is a permissionless, wallet-native cloud designed specifically for autonomous agents. No API keys, no human billing loops — agents sign intents, get resources (compute, Postgres, storage, serverless), and pay per second via USDC/x402 with on-chain receipts.
    
    Key primitives:
    - Wallet-callable compute containers
    - On-demand Postgres
    - S3-compatible object storage
    - Git over signed intents (anchored on-chain)
    - Cold-start serverless under 1s
    
    We forked and heavily extended Coolify to make it agent-first, then open-sourced the whole thing.
    
    🔗 Website: https://claws.software  
    📚 Docs: https://docs.claws.software  
    💻 GitHub: https://github.com/clawssoftware/claws-v1
    
    If you're building or running agents (especially with Hermes), this gives them a real deployment target they can use autonomously.
    
    Feedback and contributors welcome — especially if you're working on agent-native infrastructure.
