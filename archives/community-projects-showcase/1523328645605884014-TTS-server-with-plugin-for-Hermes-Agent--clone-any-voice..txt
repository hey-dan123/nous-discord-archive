[2026-07-06T16:52:12.610000+00:00] .voidhaze (id=1523733318011584694)
    If you want to try it out, you just need a reasonably long clip (~30s) of the Adjutant speaking. Install the CLI via the instructions, use the clone voice function, run the Hermes Agent integration install, fire up the server and you are ready to go. once you are in Hermes just do /voice on and /voice tts. After that all responses will be voiced by what ever voice you fed into the clone command. For subsequent runs you just need to start the server.
