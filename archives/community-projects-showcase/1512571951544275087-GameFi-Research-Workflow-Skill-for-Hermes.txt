    I wanted to explore a small Hermes-centered workflow for a gap I noticed in Web3 gaming research.
    
    Hermes already has strong research and automation workflows, but I couldn’t find a dedicated workflow for structured Web3 gaming project discovery. So I opened a small research workflow skill foundation PR.
    
    The first version focuses on:
    - early-stage project discovery workflow
    - GitHub signal analysis
    - README quality checks
    - project risk notes
    - WATCH / TEST / CONTACT / SKIP classification
    - Discord-ready daily report template
    
    I kept this first PR intentionally small: only the skill foundation and report template, so it can be reviewed easily. Future milestones can add scanner logic, scoring automation, and scheduled report generation.
