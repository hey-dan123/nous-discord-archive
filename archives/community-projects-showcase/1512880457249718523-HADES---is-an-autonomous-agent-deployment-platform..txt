    **HADES** is an autonomous agent deployment platform. The idea is simple anyone can deploy their own fully autonomous Hermes instance in under 60 seconds, directly from a web UI. No code, no server setup, no API key management required on the user's side.
    
    **Stack**
    Node.js Express backend for provisioning, Nginx + Let's Encrypt for SSL, PM2 for process management, systemd for auto-restart on reboot, and a terminal-aesthetic frontend with CRT effects and boot animation.
    
    Happy to answer any questions about the implementation!
