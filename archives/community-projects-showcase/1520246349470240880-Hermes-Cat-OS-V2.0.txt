    Im make Hermes Cat Multiagents Dashboard for controll all from Phone
    
    - Dashboard: health score, errors, API costs
    - Activity: includes workers and a Kanban board
    - Benchmark: shows the most frequently used models and their performance
    - Providers: shows which providers you use most often
    - Memory: memory, "prompt soul" memory, etc.
    - Agents: a tab with worker profiles and the option to select a model
    - Cron: includes a "trigger now" function and a history of recent runs
    - Logs: displays real-time system logs from the last 10 minutes
    - Chat: features an interactive mascot and options to select the agent model and the underlying model; updates live, allowing you to switch models within the same chat session
    
    Supports "allow once" and all tools; very fast performance (100–500ms).
