    MSLM — Multi-Scope Local Memory for Hermes Agent
    
    Ever spent an afternoon teaching your Hermes Agent your entire tech stack, only to have it forget everything the next session? Same.
    
    MSLM builds on the SuperLocalMemory engine with one key addition: three-layer scope — personal / shared / global. Your agent keeps private memories (API keys, preferences) separate from shared team knowledge (project conventions, domain facts). Multiple agents, independent personalities, shared wisdom.
    
     Local-first — Mode A runs on pure math. Zero cloud calls. EU AI Act ready.
     MCP native — hermes mcp add mslm --command mslm --args mcp
     One-liner — pip install mslm-memory && mslm setup
     4500+ tests passed — Fisher-Rao geometry, Sheaf topology, Langevin dynamics. All on CPU.
