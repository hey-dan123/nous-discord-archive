    ## Cronalytics v1.1.0 released
    
    Changelog highlights:
    
    **New interfaces:**
    - Dashboard tab with cards, leaderboard, sortable jobs table, model breakdown, and failure-state analysis
    - CLI tool — eight commands (summary, jobs, models, trends, health, runs, all, sync), `--json` output
    - Agent skill for guided cron health diagnostics with confidence-graded anomaly detection
    
    **i18n:**
    - Spanish, Simplified Chinese, Traditional Chinese — all UI, modals, and metrics
    - Multi-model 2/4 consensus validation pipeline for translation quality
    - Self-hosted catalog architecture (no Hermes framework dependency for i18n)
    
    **Quality:**
    - 149 tests, ruff clean, mypy clean
    - CI pipeline on every push/PR
    - Full documentation overhaul: FAQ, community templates, doc contracts
    
    **Install:** `pip install cronalytics` or clone into `~/.hermes/plugins/`
    **Upgrade from v1.0:** See UPGRADE.md — namespace restructured, cost fields renamed with "estimated" qualifier.
    **Docs:** https://github.com/8bit64k/cronalytics
