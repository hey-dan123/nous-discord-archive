    I built a polished Web UI for Hermes Agent
    
    Been using Hermes through Telegram and found it uncomfortable for actual work. So I built a proper web interface for it.
    
    Features:
    
    One-click setup (no Docker required)
    Multi-profile management (switch between contexts without reconfiguring)
    Settings & API keys management UI
    Multiple model support - save and switch between providers/models
    Full chat sessions with history
    Memory management (add/edit/remove memories from UI)
    Skill management (browse, load, enable/disable)
