    HyAtlas-Memory is a memory provider plugin that gives Hermes persistent, structured memory across sessions. 
    https://github.com/tuancookiez-hub/HyAtlas-Memory
    **What it does automatically**
    • Captures every conversation, extracts atomic facts via LLM
    • Injects relevant memories into the prompt before the agent responds (no tool call needed)
    • Background processing merges duplicates, resolves conflicts, refines your model over time
    **7 memory layers**
    • L1 raw (verbatim sessions) → L2 facts (LLM-extracted atomic)
    • L3 summaries (every 20 adds) → L4 identity (long-lived user facts)
    • L5 Kuzu graph (async, knowledge relationships)
    • L6 schemas (typed entity/relationship categories)
    • L7 intention (proactive intent detection — my own extension)
    **Dual-path architecture**
    • **System 1** (fast): L1/L2 in milliseconds, every message — you never wait
    • **System 2** (background): L5 graph rebuild, dedup, schema validation, intent detection
    **Live dashboard** — `python -m server.dashboard.dashboard`
    • 7 tabs: Overview, Explore, Layers, Today, Graph, Activity, Settings
    • L0–L7 layer distribution, 7-day ingestion chart, knowledge constellation
    • Reads the live server, no extra config
    **CLI** — `hermes hy-memory doctor | add | search | list | init | reset`
    **3 modes** depending on cost tolerance:
    • `lite` — embedding-only, zero LLM calls (free)
    • `pro` — LLM extraction + reconciliation
    • `ultra` — full pipeline + Kuzu graph (default)
    **Built on**
    • Hermes Agent (this org) — peer dep
    • Hy-Memory framework (Tencent Hunyuan) — architecture inspiration
    • Kuzu — embedded graph DB
    • SentenceTransformers — local embeddings (or any OpenAI-compatible embedder)
    • Works with OpenAI, OpenRouter, DeepSeek, MiniMax, ByteDance, or local Ollama
    MIT licensed. PRs welcome. The 6-layer model the System1/System2 duality comes from the Hy-Memory framework; the L7 intention layer, the dashboard's layer-aware UI, and the System2 background triggers are my additions.
