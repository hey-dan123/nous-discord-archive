    Hey everyone,
    I've been running hermes-agent for a while and absolutely love it, but when I tried to deploy some existing web UI solutions on my UGREEN NAS via Portainer to have a friendly interface to use hermes-agent from anywhere, I ran into a wall with permission nightmares, database locking, and CORS headaches... so I ended up building my own solution.
    Hermes Chat UI is a single Docker image that packages the Hermes Agent, an API proxy, and the web UI together. Pull it, set a few env vars, done. No separate agent install required. I'm running it on my NAS and accessing it remotely via Tailscale, which is great for my use case.
    Highlights:
    - Single Docker image — agent + proxy + UI, all in one
    - Mobile-first PWA, installable on your phone
    - Real-time streaming with the agent's thought process visible
    - Tool approval UI — review and approve/deny tool calls from the chat
    - Conversation persistence via SQLite
    - Home Assistant and GitHub integrations via env vars
    My goal with this post is to just share this with someone who might also need it. I'd be happy to get some feedback from you though.
    Thanks!
