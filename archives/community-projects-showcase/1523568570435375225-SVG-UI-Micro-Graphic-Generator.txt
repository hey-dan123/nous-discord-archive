    **Micro Graphic Generator**
    A procedural, zero-dependency, single-file HTML/SVG micro-graphic generator inspired by FUI/HUD design, technical line art, and abstract data visualization.
    
    **Overview**
    The Micro Graphic Generator dynamically sculpts complex, high-resolution SVG compositions on the fly using a mathematically driven procedural layout engine. It is designed to act as an endless source of inspiration or background assets for sci-fi, cyberpunk, editorial, and abstract design workflows. Every time the page is refreshed, a completely unique graphic is generated.
    
    **Features**
    Procedural Layout Engine: The core system combines base grid skeletons (Binary Space Partitioning, Concentric Rings, Cellular Automata) with radical mutators (Sparsify, Explode, Greeble, Frame) to mathematically generate thousands of unique macro-architectures.
    FUI & Tech Elements: Generates HUD crosshairs, broken targeting rings, data waveforms, circuit traces, parallel hatch shading, and perspective wireframes.
    *Cultural Aesthetics*:
    Ensō (円相): Procedural sumi-e style brush strokes and ink wash.
    Adinkra (Gyaman): Traditional West African visual symbols and geometry.
    CJK (中日韓): Generative kanji, katakana, and hanko (red seal) compositions.
    Zero Dependencies: Everything runs in vanilla JavaScript inside a single index.html file. No build steps, no external libraries, no NPM packages.
    Architecture
    The system operates in three main phases:
    
    *Palette Selection*: Chooses from a curated set of monochrome, high-contrast, or accented color palettes.
    *Macro Layout*: The layoutProcedural() engine mathematically divides the canvas into a series of structural cells using recursive algorithms.
    *Micro Generation*: Every individual cell independently picks a drawing instruction from a flat pool of 30+ generators (ranging from barcodes to mountain landscapes) and renders it using raw SVG DOM elements.
    *Usage*: Simply open index.html in any modern web browser. Refresh the page to generate a new composition.
    
    **Inspiration**
    Inspired by Chad Ashley (@cgpov), danhdox/glyph, and various Fantasy User Interface (FUI) artists.
