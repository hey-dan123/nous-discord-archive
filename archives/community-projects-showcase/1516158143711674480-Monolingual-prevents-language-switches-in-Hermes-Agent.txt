    TL;DR: Plugin that runs Lingua to auto detect a foreign language and translates when needed.
    
    I wrote this as a lightweight, no config way to ensure GLM never accidentally speaks mixed languages in user facing output. Try it out!
    
    Agent prompt:
    ```# Install monolingual Hermes plugin please!
    # Install the lingua detection library
    pip install lingua-language-detector
    
    # Clone the plugin into Hermes plugins directory
    git clone https://github.com/GwynTel/monolingual.git ~/.hermes/plugins/monolingual
    
    # Enable the plugin
    hermes plugins enable monolingual
    
    # Restart your session or gateway (or /reset in an active session)```
