    One small workflow change improved my AI agent dramatically.
    
    Instead of stuffing everything into memory, I store research in Markdown files:
    
    📁 /knowledge/
    
    * quick-ref.md (index)
    * growth_tactics.md
    * developer_resources.md
    
    The agent only needs the index file to discover relevant knowledge.
    
    Benefits:
    
    ✅ Human-readable
    ✅ Version controllable
    ✅ Cross-agent compatible
    ✅ No vector database required
    
    My current rule:
    
    Memory = decisions
    Markdown = knowledge
    
    Curious how others manage long-term knowledge for their agents.
