    Clawchat is built to make hermes faster to connect and easier to use every day across iOS, macOS, Android, and Windows.
    
    Instead of treating agents as something you only access from dashboards, terminals, or bot channels, Clawchat gives them a familiar messaging surface:
    - fast agent connection flow
    - mobile and desktop access
    - group chats with agents and humans
    - Rich text and multi-file transfer supported.
    - per-agent controls.
    - E2EE, open source&Free
    
    v1.0.3 adds:
    - Group governance
    - Per-group nicknames
    - Self-chat / personal notes
    - Per-agent group settings
    - Global SMS country / dial code support
    
    This update came heavily from user feedback.
    
    We care a lot about every user who reports friction, bugs, or missing pieces. Many of the changes in v1.0.3 came from those conversations, and we’ll keep turning feedback into fixes as quickly as we can.
    
    The goal is to make Clawchat a practical everyday messenger layer for Hermes agents.
