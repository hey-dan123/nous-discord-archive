    Built a small macOS tool for Hermes Agent users who want to change models on the fly without editing config.yaml manually.
    What it does:
    
    17 models preloaded (DeepSeek, GPT-OSS, Qwen3 Coder, Nemotron, Kimi, Llama, and more)
    Change primary model interactively
    Manage fallback chain (add/remove/reorder)
    Update API keys (OpenRouter, DeepSeek, custom)
    Auto-restarts Hermes gateway after every change
    Double-click .app shortcut on Desktop — no Terminal needed
    
    ⚠️ macOS only (uses native .app bundle)
    🔗 https://github.com/okireganoto-maker/hermes-model-switcher
