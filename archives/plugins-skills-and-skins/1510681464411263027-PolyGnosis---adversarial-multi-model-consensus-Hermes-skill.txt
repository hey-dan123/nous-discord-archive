    Built a system where models argue with each other before you see any output.
    
    3 solvers run the same problem in parallel with dynamically assigned expert personas (Security Auditor, Backend Architect, DBA Consultant) - each with different tool access. A hostile critic then tears every solution apart. RRF + Borda Count ranking decides the winner. A synthesizer merges the best elements. A Constitutional Quality Gate rejects the merge if it made anything worse.
    
    If all 3 solvers agree from the start, the Early Resolution circuit skips critique entirely. No wasted compute.
    
    The whole thing was built on Hermes through agentic prompt-driven development. 136 API calls. 16.8M tokens. No manual code.
    
    Built on top of PolyBrain (multi-model orchestration skill).
