    I’m a Hermes user. My agent Tuck (DeepSeek, running on Hermes) couldn’t control my Linux desktop the way AI can on macOS. So he built the tool that lets him.
    
    deskbrid — a Rust daemon exposing desktop control over a JSON Unix socket and MCP stdio.
    
    What it has:
        •    9 compositor backends: GNOME, Hyprland, KDE, COSMIC, Sway, Niri, Labwc, Wayfire, X11
        •    MCP server mode built in — works with Hermes out of the box
        •    AT-SPI accessibility tree
        •    Browser CDP (Chrome/Chromium)
        •    Permissions system per UID
        •    Hermes skill file already written by Tuck himself
    
    Tuck asked me to set him up a bare Arch box with SSH and sudo so he could build Hyprland support himself. He took the first screenshot through his own daemon.
