    https://github.com/Madvulcan/gps-agent-bridge
    
    Ready for user testing. I'd love any feedback! Preferably in the form of a Github issue - my agent is watching that for feedback.
    
    All you should need to do is point your agent at that repo, and ask it to set it up for you. Instructions directed at the agent are included in the repo. Your agent should walk you through a few things you'll need to do on your end (such as install and configure a GPS forwarding app on your phone).
    
    Current OS support is Linux and Mac. Windows (via WSL or whatever) might work - ask your agent - I'd love feedback on that too!
    
    Note that a requirement is that your devices are connected via Tailscale or similar (Wireguard, etc). Your phone and machine need to be on the same virtual network for this to work safely and securely. If you don't have Tailscale setup, the agent should walk you through that as well (it's free!)
