    Made a small Hermes plugin: a HashiCorp Vault secret source.
    
    Right now, to give the agent its keys and passwords you have to put them into a .env file on every machine it runs on. That gets messy fast, and plaintext secrets sitting on disk are an easy thing to leak. If your team already keeps secrets in Vault, you were copying them out by hand anyway.
    
    This lets you keep everything in Vault instead. The agent reads one Vault path on startup and loads every key there as an env var, same idea as the built-in Bitwarden source. Change a secret once and every agent gets the new value, nothing sensitive left lying in files. Works with token or AppRole login, TLS on by default.
    
    pip install hermes-vault-secret-source
    hermes plugins enable hermes-hashicorp-vault
