    Hey Fam👋
    
    I built a small plugin that might be useful for anyone doing real email workflows with Hermes:
    
    https://github.com/artguerilla/hermes-multimail
    
    The default Hermes mail setup works fine for one account, but I needed Hermes to work across multiple mailboxes. The preinstalled himalaya setup also didn’t cover attachments for my use case, which was a blocker because a lot of invoices come in as PDFs.
    
    So I made **hermes-multimail**.
    
    My original use case was pretty simple: collect invoices from multiple email accounts, read/search them from one Hermes agent, download the attachments, and then organize them properly.
    
    Current features:
    
    - multiple IMAP/SMTP accounts
    - search across one or all inboxes
    - read full emails
    - attachment detection/download
    - send emails
    - reply in-thread
    - list folders
    - mark messages as read
    - move/delete messages
    
    It’s just a generic Hermes plugin, no core Hermes changes needed. Drop it into the plugins folder, configure your accounts, and Hermes gets multi-mailbox tools.
    
    Would love feedback from anyone else using Hermes for inbox automation, invoice handling, or multi-account assistant workflows.
