    Following the standalone-plugin guidance on Hermes Agent issue #10703, I published a Deepgram voice plugin for Hermes Agent:
    
    https://github.com/jaysonsantos/hermes-deepgram-voice-plugin
    
    It provides:
    • Deepgram Aura text-to-speech
    • Deepgram Nova speech-to-text
    • Hermes provider registration without core changes
    • Configurable provider/model options
    
    Install:
    hermes plugins install jaysonsantos/hermes-deepgram-voice-plugin --enable
    
    Set DEEPGRAM_API_KEY, then configure the Deepgram provider in Hermes.
    
    Quality: 32 tests passed (93.75% coverage), Ruff, pre-commit, package build/Twine validation, actual Hermes installer smoke test, and GitHub Actions all pass.
