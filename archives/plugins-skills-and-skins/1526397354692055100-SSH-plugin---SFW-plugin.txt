    I’ve started publishing a couple of small Hermes plugins under the hermes-plugins repo.
    
    Hermes SSH gives agents a cleaner way to manage SSH machines, run commands, and keep track of sessions without repeatedly handling connection details.
    
    Hermes SFW routes package installs through Socket Firewall Free, adding a supply-chain check before an agent installs dependencies with npm, pnpm, pip, uv, or cargo.
    
    Both are early, but they’re already useful guardrails for agents operating against real systems.

    - Hermes SSH on PyPI: https://pypi.org/project/hermes-ssh/  
    - Hermes SFW on PyPI: https://pypi.org/project/hermes-sfw/
