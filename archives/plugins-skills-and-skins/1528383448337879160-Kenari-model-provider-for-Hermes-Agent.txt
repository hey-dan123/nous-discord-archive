    I published a standalone Kenari provider plugin for Hermes Agent. Kenari is an OpenAI-compatible LLM gateway.
    
      Features:
      • live model and context-length discovery
      • live IDR pricing metadata, with no static prices or zero placeholders
      • tested against Hermes Agent 0.18.2 and current main
      • supports Python 3.11, 3.12, and 3.13
    
      Repository:
      https://github.com/kenarihq/hermes-kenari-provider
    
      Release:
      https://github.com/kenarihq/hermes-kenari-provider/releases/tag/v1.0.0
    
      Install:
      git clone https://github.com/kenarihq/hermes-kenari-provider.git ~/.hermes/plugins/model-providers/kenari
    
      Set KENARI_API_KEY, restart Hermes Agent, then select Kenari as the provider.
    
      Related provider-neutral pricing PR:
      https://github.com/NousResearch/hermes-agent/pull/67489
