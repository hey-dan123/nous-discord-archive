    github-pr-reviewer skill — PR: https://github.com/NousResearch/hermes-agent/pull/40530
    
    Automated code review for any GitHub pull request. Give Hermes a PR URL 
    and it fetches the diff, reviews every changed file across five dimensions, 
    then posts structured inline comments directly to the PR.
    
    What it reviews:
    - Bugs (null dereference, off-by-one, race conditions)
    - Security (SQL injection, hardcoded secrets, missing auth checks, insecure defaults)
    - Error handling (silent except blocks, unclosed resources, swallowed errors)
    - Performance (N+1 queries, blocking I/O in async context)
    - Test coverage (new endpoints or functions with no tests)
    
    Trigger: "review PR https://github.com/owner/repo/pull/42"
    
    Requirements: gh CLI authenticated (or GITHUB_TOKEN env var as fallback).
    Safe by default — posts comments only, never auto-approves or requests 
    changes without explicit confirmation. Skips binary, minified, and 
    generated files automatically.
