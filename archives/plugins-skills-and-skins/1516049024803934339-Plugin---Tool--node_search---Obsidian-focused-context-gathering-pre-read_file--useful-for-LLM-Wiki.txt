    node-search is a Hermes plugin for bounded Markdown/Obsidian context gathering before read_file.
    
    It indexes frontmatter, headings, wikilinks, backlinks, graph-health states, and returns compact candidate evidence instead of dumping files.
    Useful for vault/wiki workflows where agents need to choose the right notes safely and cheaply.
    Configurable roots, path confinement, CI, tests, and synthetic benchmarks included.
