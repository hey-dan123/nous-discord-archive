    A drop-in dashboard plugin that turns your Hermes sessions into real analytics — built to go beyond the native Analytics page.
    
    - **Cost attribution** — see exactly who and what is spending: per user, per platform (webhooks + crons), and per model, plus your top-N most expensive sessions.
    - **Tool insights** — global and per-session tool usage with average latency, so you can spot slow or chatty tools.
    - **Cleaner conversations** — strips the gateway's thread-context / reply / sender prefixes into a readable transcript, with an "Open in Slack" deep link and the skills each session triggered.
    - **Flexible time ranges** — presets plus custom since/until windows, not just fixed 7/30/90 days.
    - **Safe by design** — reads `state.db` read-only, never calls an LLM.
    
    Currently focused on Slack; Discord and WhatsApp attribution coming soon. Install = clone into `~/.hermes/plugins/` and reload the dashboard.
