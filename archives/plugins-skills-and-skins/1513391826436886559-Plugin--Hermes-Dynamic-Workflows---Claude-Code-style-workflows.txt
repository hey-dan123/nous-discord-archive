    Hey all — I built an open-source plugin that brings Claude Code-style Dynamic Workflows to Hermes 👋
    You tell Hermes "run a workflow that …" and it writes a sandboxed Python script that orchestrates subagents with agent()/parallel()/pipeline() — parallel fan-out + adversarial verify, up to 1000 subagents. Comes with persisted/resumable runs, a live dashboard, and a /workflows history view.
    One-line install: hermes plugins install lingjiuu/hermes-dynamic-workflows --enable
    Repo: https://github.com/lingjiuu/hermes-dynamic-workflows
