    I’m sharing my standalone DeepThink skill handoff as a PDF + Markdown file.
    
    DeepThink is a reusable agent skill, not a model or plugin. It gives Hermes a structured loop-based reasoning workflow for moments where a quick answer is too shallow.
    
    It runs through:
    
    - Surface: first-pass hypothesis
    - Explore: alternatives, evidence, edge cases
    - Challenge: assumptions, risks, failure modes
    - Synthesize: refined answer and convergence check
    
    Use it for architecture decisions, security analysis, strategy, tradeoffs, comparing approaches, and “what am I missing?” moments.
    
    Install target for Hermes:
    
    `~/.hermes/skills/deepthink/SKILL.md`
