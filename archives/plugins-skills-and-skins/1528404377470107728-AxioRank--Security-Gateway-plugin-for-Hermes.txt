    🛡️ hermes-axiorank routes every Hermes tool call and tool result through AxioRank, the security gateway for AI agents. It scores actions for live secrets, prompt injection, destructive operations, PII, and risky egress, offline and keyless by default.
    
    What it does:
    • 3 guard tools the agent can call: `axiorank_inspect`, `axiorank_scan_mcp` (verify an MCP server or agent card), `axiorank_verify`
    • Injects a live guardrail policy into the model turn, plus a one-shot taint warning when a tool result is poisoned (agentjacking defense)
    • Optional hard block: `hermes axiorank init` installs a pre_tool_call shell hook that denies a dangerous action before it runs
    • Bundled `secure-tool-use` skill; per-session audit and a signed Coding Session Seal with a key
    
    Install:
    `pip install hermes-axiorank && hermes plugins enable axiorank`
    Works with no key (local detectors). Set `AXIORANK_API_KEY` to apply your workspace policy and central audit.
