    I made `potaco` skill for agents.
    
    It helps agents use the `potaco` CLI safely for image generation and editing. The skill covers:
    
    - Text-to-image generation
    - Image editing, inpainting, and outpainting
    - OpenAI, fal, Vercel AI Gateway, and custom OpenAI-compatible providers
    - Model discovery and provider setup guidance
    - Troubleshooting CLI failures without leaking credentials
    
    The main thing I wanted was a safer way for agents to work with image providers. The skill explicitly avoids asking users to paste API keys into chat, prefers interactive auth, checks `potaco version` before doing anything, and treats prompts, image paths, base URLs, and debug logs as sensitive.
    
    Install: `npx skills add ncxton/potaco`
    Repo: https://github.com/ncxton/potaco
