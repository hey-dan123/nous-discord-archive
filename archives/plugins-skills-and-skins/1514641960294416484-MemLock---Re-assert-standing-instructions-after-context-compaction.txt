    MemLock — Your Agent Stops Forgetting Instructions
    
        You know that thing where you tell your agent "always use bullet points" and 30 turns later it's writing you essays? That's context compaction eating your instructions.
    
        MemLock is a Hermes plugin I built that fixes it. When the system compresses old turns into a summary (to save tokens), MemLock watches for that moment and checks: did your pinned instructions survive in the active conversation, or did they get buried in the summary?
    
        If they drifted, it quietly re-injects them so your agent picks the instruction back up. No drama, no lost context.
    
        You pin instructions via guard_pin — things like "always reply in British English" or "use short sentences". Run /guard anytime to see how healthy your standing instructions are.
    
        Setup is one git clone into your plugins folder. MIT licensed, tested, docs included.
    
        Repo: https://github.com/Sahil-SS9/hermes-memlock

    Just shipped a solid update to the memlock plugin (the thing that stops Hermes forgetting your pinned instructions after context compaction).
    
        What's new:
    
        scope: global — pins that survive session restarts. Set it once, it comes back every session. Default is still session (dies with the session), so nothing breaks.
    
        python
        guard_pin(text="Always use British English", scope="global", priority=80)
    
    
        Also in this release:
        - inject: always mode — deterministic presence every turn if you want it
        - Pin cap (max_pins: 16) — no prompt-injection pin spam
        - Whitespace-flattened pin text — can't spoof extra lines in the reminder block
        - Windowed semantic audit — per-window max cosine instead of whole-region averaging (way more accurate for short instructions)
        - Memory-provider agnostic persistence — defaults to zero-dependency file store, pluggable for Mnemosyne later
    
        Upgrade:
        bash
        git pull origin main  # if you cloned it
        or
        git clone https://github.com/Sahil-SS9/hermes-memlock.git ~/.hermes/plugins/memlock
