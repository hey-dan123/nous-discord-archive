    MemLock — Your Agent Stops Forgetting Instructions
    
        You know that thing where you tell your agent "always use bullet points" and 30 turns later it's writing you essays? That's context compaction eating your instructions.
    
        MemLock is a Hermes plugin I built that fixes it. When the system compresses old turns into a summary (to save tokens), MemLock watches for that moment and checks: did your pinned instructions survive in the active conversation, or did they get buried in the summary?
    
        If they drifted, it quietly re-injects them so your agent picks the instruction back up. No drama, no lost context.
    
        You pin instructions via guard_pin — things like "always reply in British English" or "use short sentences". Run /guard anytime to see how healthy your standing instructions are.
    
        Setup is one git clone into your plugins folder. MIT licensed, tested, docs included.
