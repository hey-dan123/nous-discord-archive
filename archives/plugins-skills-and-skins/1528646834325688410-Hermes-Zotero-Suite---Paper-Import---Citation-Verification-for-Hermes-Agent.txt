    # Hermes Zotero Suite — Paper Import & Citation Verification for Hermes Agent
    
    Four scripts, one Zotero backend. Covers the full pipeline from paper discovery to pre-submission citation audit.
    
    **📥 Three Import Paths**
    - `arxiv_to_zotero.py` — batch import by arXiv ID, auto-fetches metadata + PDF
    - `doi_to_zotero.py` — DOI import, works with Nature / PNAS / Elsevier / APA / ACM / IEEE
    - `web_to_zotero.py` — blog posts, open-source repos, documentation sites as Zotero items
    
    **🔍 Citation Verification**
    - `verify_citations.py` — scans a .tex bibliography, resolves every DOI/arXiv ID through Zotero, flags year mismatches and broken references
    
    **Wired into refceris** — the verification script plugs into refceris' 5-step citation audit as a fourth data source alongside arXiv API / CrossRef / Semantic Scholar, with dual-source cross-validation.
    
    GitHub: https://github.com/nuffin/hermes-zotero-suite
    Requires: Zotero running + local-write-api plugin v3.3.1+
