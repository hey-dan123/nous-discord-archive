    I’ve published a standalone Hermes Agent plugin for DaVinci Resolve, following the repository’s standalone-plugin guidance:
    
    https://github.com/wassermanproductions/hermes-davinci-resolve-plugin
    
    It supports Resolve Studio live control plus free-edition FCPXML and marker CSV workflows. The reviewed hardening is included: package-relative imports with no global sys.path mutation, dry-run-safe interchange defaults, macOS check_fn gating on every tool, and a real PluginManager user-install discovery/registry regression test against current Nous main. Public CI is green, and v0.3.0 is available here:
    
    https://github.com/wassermanproductions/hermes-davinci-resolve-plugin/releases/tag/v0.3.0
    
    Install:
    ```bash
    git clone https://github.com/wassermanproductions/hermes-davinci-resolve-plugin.git ~/.hermes/plugins/davinci-resolve
    hermes plugins enable davinci-resolve
    hermes plugins list
    ```
    
    Feedback is very welcome. Thanks again to Teknium for the review and direction.
