    A small thinking layer plugin i put together for Hermes. Nothing too ambitious, just gets the job done. Small steps 🙂
    
    • Detects user goals (Information / Understanding / Action) • Generates multiple scenarios before answering • Runs Monte Carlo simulation (10K-50K iterations, 0 LLM tokens) • Shows a [DOGA: Thinking Process] panel in responses
    
    Depth 1-5, /doga slash commands, simulate tool included. Plugin-based, no core changes. Any feedback appreciated.
