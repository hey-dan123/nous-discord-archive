    🧠 Hermes Memory Plugin
    
    > "Your AI forgets everything. Hermes remembers."
    
        Hermes Agent stores memory in flat .md files — no search, no embeddings, no context.
    
        The Overmind plugin replaces that with a persistent semantic memory layer:
    
    > ✅ 4096D Vector Search — Qwen 8B embeddings via OpenRouter + PostgreSQL/pgvector
    > ✅ Per-agent isolation — Each agent gets its own private database
    > ✅ Automatic — Stores every exchange, prefetches relevant context before each response
    > ✅ Explicit tools — memory_store, memory_search, memory_runs
    
        One line in config.yaml to activate. No more lost context between sessions.
