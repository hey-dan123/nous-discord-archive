    People thought we were joking when we said this could actually work — but here it is.
    
    RIFTAGENT is an open-source LLM workflow that turns a simple theme (weapon, mercenary, companion, shinobi, island — whatever you throw at it) into a fully described, studio-quality item drop. Every roll pulls from Legendary, Mythic, or Secret/Ultimate tier only, equal odds, no filler commons. Each item gets a unique LLM-written description, a formatted image prompt, and a provenance record so you always know exactly where it came from.
    
    What's in the repo:
    
    📄 README — full concept + install/usage overview
    🧩 SKILL.md — agent-loadable instructions (agentskills.io ready)
    ⚙️ SPEC.md — full technical spec: schemas, prompt templates, backend contract
    
    It's designed to be backend-agnostic — ships with Nano Banana as the default renderer, but the orchestration logic (theme → tier roll → item gen → prompt) is fully decoupled, so it can run local (Stable Diffusion/ComfyUI) or through Hermes natively later.
    
    MIT licensed. Fork it, extend it, build your own gacha/marketplace/game on top of it.
    
    🔗 https://www.github.com/ohrbit/riftagent
    
    If you build something with it, tag us — we'd love to see what comes out of the rift. 🔥
